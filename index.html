Queercraft is a Minecraft Server accessible by Java clients running version 1.21.7 (or later)

TLDR:

There are certain mods which allow you to bypass server blacklists

qConnect happens to be one of those mods

qConnect requires the Fabric (or Quilt) Mod Loader (odds are you already have this)

qConnect Mod: Direct Download (from the qConnect Repository on Degghre's GitHub)

Simply place this .jar file into your Minecraft's "mod" folder with the rest of your mods

That is it :)















Running the standard Minecraft Launcher from minecraft.net?
Hit your Windows Key and R to get the run prompt, type %appdata% then click OK
open the .minecraft folder then open the mods folder and place the qConnect mod there

Don't see a mods folder in the .minecraft folder? You need the Fabric Modloader
Fabric Modloader: Go here to download and install if you have the standard Minecraft Launcher
